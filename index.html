Objeto documento(DOM) (document)

Permite al javascript modificar y manipular dinámicamente el contenido, la estructura y el estilo de un documento HTML.
Ejemplo: Si no tenemos acceso al html podemos modificar los estilos como el color, y lo podemos hacer mediante el javascript.

- Obtener un Elemento: getElementById

Se utiliza para obtener una referencia a un elemento HTML utilizando su atributo id. Este método permite a los desarrolladores acceder y manipular dinámicamente el contenido y los estilos de un elemento específico en un HTML.
EJEMPLO: Acceder a un elemento "p", que previamente se le asigno un ID "message"

- Obtener un Elemento: querySelector

Al igual que el getElementById tambien se utiliza para obtener una referencia a un elemento HTML, pero en esta ocasión se puede acceder a el mediante un ID(#) o mediante una clase(.)
EJEMPLO: Acceder a un elemento "p", que previamente se le asigno una clase "message"

- Obtener Elementos: querySelectorAll

Al igual que los anteriores getElementById querySelector tambien se utiliza para obtener una referencia a un elemento HTML, pero en esta ocasión se puede acceder a varios elementos mediante un ID(#) o mediante una clase(.)
EJEMPLO: Acceder a las etiquetas "a" de una lista de la navegacion, que previamente se le asigno una clase "navBar"

Nodos( que cosa son los nodos )

los nodos son los elementos fundamentales que conforman la estructura de un documento HTML. El DOM organiza la estructura de un documento como un árbol de nodos, donde cada elemento, atributo, texto, comentario, etc., se representa como un nodo en el árbol.
EJEMPLO: Nodo de texto, nodo de elemento, nodo de atributos, etc.
Otro dato importante es que todo elemento HTML es un nodo, pero no todo nodo es un elemento HTML.

TextContent (contenido elemento)

es una propiedad en el DOM (Document Object Model) de JavaScript que proporciona o establece el contenido de texto de un nodo y todos sus descendientes. Esta propiedad incluye todo el texto contenido en el nodo, incluyendo el texto de todos sus hijos.
EJEMPLO, Obtener acceso a un elemento mediante un getElementById("miDiv"), luego obtener el contenido con un = miDiv.textContent, y finalmente mostarlo mediante un console.log.

estilos

Otra caracteristica del DOM es que nos permite cambiar estilos de los elementos.
EJEMPLO: message.style.color = 'red'

DomTraversing

Esta propiedad del DOM permite recorrer de un elemento hijo hacia un elemento padre
EJEMPLO, Movernos desde una etiqueta "a" hacia una etiqueta "nav", esto lo hacemos con un "parentElement".

createElement

El DOM tambien nos permite crear elementos que no han sido creados en el html
EJEMPLO: Esto lo hacemos con el createElement

innerHTML

permite acceder y manipular el contenido HTML dentro de un elemento. Puedes obtener el contenido HTML de un elemento o establecer un nuevo contenido HTML utilizando esta propiedad.
EJEMPLO: crear un innerHTML = 'h1'
